# Task2: Heap and Heap Sort

## Data Structure: Min Heap
- Definition: A complete binary tree where the value of the parent node is less than or equal to that of the child nodes.
- Application Scenarios: Priority queue, heap sorting. 

## Algorithm: Heap Sort
- Idea: Utilize the heap structure to repeatedly extract the smallest (largest) value to achieve sorting.
- Time Complexity: O(n log n)
