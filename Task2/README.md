# Heap Data Structure and Heap Sort Algorithm

## Project Description

Self studying a new data structure Heap and a new algorithm Heap_Sort

Heap is a special type of complete binary tree which satisfies the heap property.  
Heap Sort is a comparison-based sorting algorithm developed using the Heap data structure.


## Project Structure

This folder contains the following Python files:
- heap.py
- heap_sort.py


## Requirements

- Python 3.x installed on your system
