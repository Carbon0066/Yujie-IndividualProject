# Restaurant Seat Reservation System

## Overview

This project is an Object-Oriented Programming (OOP), using Python to developed.  

The system is designed to simulate real-life restaurant table booking system, allowing admin to manage customers, tables, and reservations efficiently. Use a command-line interface (CLI) as User Interface Design.

The main features of this system include:

- Add new customers
- Add new tables
- Automatically assign tables for customers according to the time and capacity
- Make reservations
  - default dining time is 90 minutes
- Cancel reservations
- View customer booking history (including cancelled bookings)
- Prevent double booking through time conflict checking
- Automatically assign table location:
  - Tables with capacity 8 or above : VIP Room
  - Tables with capacity below 8 : Hall


## Project Structure

This project follows a modular programming approach and is divided into 5 Python files:
- customer.py        
- table.py           
- reservation.py     
- system.py          
- main.py
